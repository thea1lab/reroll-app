# Ricetta App

Mobile app to organize recipe groups and pick what to cook.

## Run

```bash
npm install
npx expo start
```

## Main Screens

- Groups
- Group Recipes
- Recipe Details
- Ricetta Modal

## Tech

- Expo + React Native
- Expo Router
- Firebase Auth + Firestore
- TypeScript

## Project Structure

- `app/` routes and screens
- `components/` reusable UI
- `storage/` Firestore + data context
- `contexts/` auth and language state
- `constants/` shared types/theme
